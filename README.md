# 📊 BCG Gen AI Job Simulation (Forage)

This repository contains my work from the **BCG Gen AI Virtual Job Simulation** on Forage. The project focuses on extracting and analyzing financial data from real-world company filings to generate insights and simulate building an AI-powered financial assistant.

---

## 🚀 Project Overview

In this simulation, I worked on:
- Extracting financial data from 10-K filings of  
  - Microsoft  
  - Tesla  
  - Apple  
- Organizing multi-year financial data (revenue, profit, etc.)
- Performing trend analysis
- Generating insights using Python
- Applying findings toward an AI-driven financial chatbot

---

## 🧠 Key Learnings

- Real-world financial data is messy → manual extraction matters  
- Trend analysis is more valuable than raw numbers  
- Business context > just technical analysis  
- Translating data into insights is the actual skill  

---
